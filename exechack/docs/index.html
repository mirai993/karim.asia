Popular problems
"MSVCP140.dll code could not be found" or "0xc00000ba"
- download visual c++, directx

loader "game not found, running"
- dont use x64 loader for x32 game or vice versa, remove "-multirun" from launch option

Loader "Injecting.." and nothing
- run loader by administrator
- use vpn, maybe in your country website banned
- try run loader 20 times in a row
- try install all visual c++
- try change the version of the game and loader to another

Loader "Downloading.." and nothing
- load with vpn
- disable dpi bypasser, he interferes with the loader protection.

I couldnt log into my account and I havent logged into it for a long time
- auto deleted dead accounts havent used it for more than half a year, to restore, create a ticket with a request to restore and present: login + password + email about purchase + purchase receipt from bank.

where i can reset password?
- in password and security, or logout and click forgot password

not working custom zoom, how fix it?
- cheat not support server zoom from weapons and thirdperson and etc

where to know is where exechack is detected?
- exechack.cc/serv

screen twitching on server
- enable Aimbot>Main>Camera Fix

lower fps or breaked esp
- perhaps something went wrong with your settings, try the config of other users from the forum
- check Config>Optimization tips

"Failed lua initialization" or "The game crashes with an error ("gamemode" library is not a table)"
- Steam>Garry's Mod>Local Files and press on "check integrity of game files"
- delete folder lua in Steam/steamapps/common/GarrysMod/garrysmod/cache
- possible server using addon gluapack, on some servers it is not compatible with the exechack, need disable gluapack.
- maybe current Run Method not suitable for this server(method of injection in server addons), try all the others in Cheat>General>Run Method
- report about this server in tickets if all of the above did not help

cheat miss in hvh (hack vs hack)
- exechack not created for hvh and not have competitive hvh functionality, search in lua list cheat for hvh
